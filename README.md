# Cosmic Clicker
A clicker game where the only goal is to cause absolute chaos.

Installation:
Make sure you have Python 3.7+ installed (download from python.org).

Install Pygame (the only external dependency):
pip install pygame

Download the script – save CosmicClicker.py to a folder of your choice.

Run the game:
python CosmicClicker.py

or Right click -> Run with Python
